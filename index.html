Ironsworn Oracle collection

With massive thanks to the data entry work done by rsek for the datasworn repository on Github, without whom I just wouldn't have done this.

Click an oracle while holding shift to have the weightings of the oracle table logged in the console. This may also be somewhat useful if you want to copy the output of an oracle (as clicking with shift does not re-roll).

Click the oracle while holding the alt/option key if you want to force a different option to be selected.
